## Features
- Fully Responsive
- Dark + Light Mode
- Smooth Scrolling
- Mobile Menu
- Ready-to-use sections (Hero, Features, Pricing, Testimonials, FAQ, Footer)

## How to Customize
1. Open `index.html`
2. Change the text, colors, and images
3. Replace logo and screenshots
4. Update links in the navbar

## Color Customization
Edit the Tailwind config's `pulse` color palette in the `<script>` tag at the top of `index.html`, or adjust the CSS variables if using plain CSS.

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
